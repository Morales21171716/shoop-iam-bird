# **App Name**: DirectBuy Listings

## Core Features:

- Product Listings: Display products with images, prices, and key features in a clear list format.
- Telegram Buy Button: Each product listing includes a 'Buy' button that directly links to the seller's Telegram.

## Style Guidelines:

- Primary color: Soft Lavender (#D0C9E2) to create a soothing and reliable environment for purchase decisions.
- Background color: Light gray (#F5F5F5), very close in hue to the primary color but greatly desaturated and light, providing a neutral backdrop.
- Accent color: Dusty Purple (#A299B0), analogous to lavender but slightly less saturated and lighter to indicate CTAs and highlights.
- Font: 'Inter' sans-serif for a clean and modern display of product details.
- Simple, minimalist icons for product categories and features.
- Clean, grid-based layout to showcase products effectively. Product descriptions are short and clear.
